# Neuronal-Partitioning-of-Spatial-Domains
Based on neural networks, we implement weight assignment via spatial allocation, giving each neuron its own receptive field for faster convergence. This method effectively simulates smooth functions and physical rules. 

We propose an improved neural network structure that builds upon the traditional multi-layer perceptron (MLP) by introducing a weight responsibility allocation mechanism. This mechanism employs a local allocation strategy based on proportional probability to achieve efficient utilization of network parameters. Specifically, the responsibility of each neuron connection is determined by the weight factor \frac{1}{(x - w_i)^2 + 1}, which is calculated based on the distance between the input x and the learnable center point w_i, endowing the network with local sensitivity to the input space. To prevent potential numerical instability during chain propagation, the design does not introduce additional importance parameters. Instead, it implicitly enhances the model's ability to express task importance by increasing the depth or width of the network. For neuron groups corresponding to the same output dimension, the responsibility (i.e., the weight factor) of each connection is computed separately and further locally differentiated, thereby improving the network's capability to model local patterns in the data. Extensive experimental results demonstrate that this network significantly improves performance across various function fitting tasks, achieving better fitting results on most datasets compared to standard MLPs.
